# EaseCation JE 测试服 Mod 包

## 环境要求

- **Minecraft Java Edition** 1.21.11
- **Fabric Loader** 0.18.4 或更高版本

## 安装方法

### 方法一：使用整合包（推荐）

下载整合包 (.zip) 文件，在 HMCL 或其他支持 CurseForge 格式的启动器中导入即可自动安装。

### 方法二：手动安装

1. 安装 [Fabric Loader](https://fabricmc.net/use/installer/)，选择 Minecraft **1.21.11** 版本
2. 将 `mods/` 文件夹中的所有 `.jar` 文件复制到 `.minecraft/mods/` 目录
3. 将 `config/` 文件夹复制到 `.minecraft/config/` 目录

### 文件夹路径

| 系统 | .minecraft 路径 |
|------|----------------|
| Windows | `%appdata%\.minecraft\` |
| macOS | `~/Library/Application Support/minecraft/` |
| Linux | `~/.minecraft/` |

## 包含的 Mod

| Mod | 说明 |
|-----|------|
| bedrock-loader | Bedrock 加载器（核心） |
| fabric-api | Fabric API 基础库 |
| fabric-language-kotlin | Kotlin 语言支持 |
| viabedrockutility | ViaBedrock 工具 |
| ImmediatelyFast | 渲染性能优化 |
| moduiclient | UI 客户端 |
| multilineactionbar | 多行操作栏 |
